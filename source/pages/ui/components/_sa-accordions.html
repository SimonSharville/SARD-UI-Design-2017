Accordions and Collapsable Panels

There is very little differnce between an Accordion and Collapsable Panels other than an Accordion has data-parent="#accordion" added to the heading link. This causes other panels within the panel-group to close when one is open, behaving more like an accordion. A Collapsable Panel can have more than panel open at a time.

There has been some CSS changes to the standard Bootstrap version, so that the entire panel-heading is a link and has roleover states.

To add the + or - icons, Add <i class="more-less glyphicon glyphicon-plus pull-right"></i> within the link, and change it to either a plus or minus depending on it's opening state.

The containing panel-group will need the class sa-accordion added to it.

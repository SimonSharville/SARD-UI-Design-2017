Tables

Tables should always sit within Panels, even if those panels have no heading or footer. They will however need to sit within a Panel Body.

 

Best Practice

Full width Table within Panels

To make the table to extend the full width of the panel then you should ADD a class of .panel-table to the Panel-Body that contains that table.

Optional

Non full width Table within Panels

Occasional you may wish to have the table not fit the full width of the panel body. to do this simple do NOT add the .panel-table to the Panel-Body that contains that table. This will revert to bootstraps default settings.

Responsive Tables

To make the table responsive the table needs to be wrapped with a .table-responsive class.
In most cases that class should be added to the containing panel. i.e:
<div class="panel panel-default table-responsive">.

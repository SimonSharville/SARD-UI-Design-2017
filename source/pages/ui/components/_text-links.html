Text Links

To include Link Icons before the text link so that the icon hangs properly on multi-line links, Add the class class="link-icon".

When in a List within a Panel, it can be added to the UL tag <ul class="link-icon list-group"></ul>.

When in a Table it should be added to the individual TD tag, and the link should be wrapped in a p tag,<td class="link-icon"></td>

In all cases the link should be wrapped in a p tag. ie:
<ul class="link-icon list-group">
    <li><p><a href="#">List Link</a></p></li>
</ul>
or
<td class="link-icon"><p><a href="#">Table Link</a></p>

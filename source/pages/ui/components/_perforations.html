Full width perforations

Perforations are written as<div class="row perfs"></div>.
To add a perforation that fits full width of the page, it should sit outside of a container.

Standard perforations

To make a perforation fit the width of the container, it should sit inside of that container.

Perforations between coloured rows

So far we rarely use this but if you wish to have rows with a background colour, you should add the colour to the row class. i.e.<div class="row row-turq"></div>. Then if you wish to add perforations between those rows you should add the class "row-top-perfs" to the top row, and a class of "row-bottom-perfs" to the bottom row. If there are multiple rows with perforarions then add both classes to the middle rows.

<div class="row row-grey row-bottom-perfs"></div>

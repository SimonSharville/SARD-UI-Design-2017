SARD Nav Tabs

SARD Nav Tabs come in either Horizontal or Vertical format.

Vertical Tabs can be nested within Horizontal Tabs or used independently.

Horizontal Tabs can be nested within Vertical Tabs or used independently.

Horizontal tabs can be standard width or justified.

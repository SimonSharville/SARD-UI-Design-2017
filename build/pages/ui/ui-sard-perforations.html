Rows & Perforations

Everything is contained within ROWs <div class="row"></div>. If you wish to add a perfration between 2 ROWs then simply add the class "row-top-perfs" to the top row, and a class of "row-bottom-perfs" to the bottom row. If there are multiple rows with perforarions then and both calsses to the middle rows.

 

If you wish to have a coloured rows then add the class of "row-grey" or "row-turq".
This row is written:
<div class="row row-grey row-bottom-perfs"></div>

This row is written:
<div class="row row-turq row-top-perfs row-bottom-perfs"></div>

This row is written:
<div class="row row-grey row-top-perfs"></div>

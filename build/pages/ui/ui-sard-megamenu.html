Megamenu

Click on the link in the top nav to view it.

To change it's colour to the admin colour, simply add admin after menu-large
i.e class="dropdown menu-large admin"

If you don't want the dropdown menus, including the megamenu to appear on hover, then de-activete the JS script in javascripts/sard/sa-megamenu.js

Messages

.message, nested within a.message-group, A message can be collapsable or not, but you shouldn't put collapsable and non-collapsable within the same group. You can however have multiple groups.
When used without a panel they should all be wrapped in a DIV with a class of .messages. Regardless of whether they are within a panel or not, the type should use the classes set out below.

To include the + / - symbols in the collapsable messages, simply add .toggle-plus to the .accordion-toggle class.

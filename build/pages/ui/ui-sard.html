Status Alerts

This shows Status Alerts within a list.
They can be within a Panel, Table or both.

Status alerts are designed to show the status of a document in a list by colour coding.
They follow the same principles of the Bootstrap alerts but also have their status initial within them.
Success = Green
Warning = Yellow
Danger - Red

We have also added
Alert = Orange
None = Clear

Panel Status Alerts

When creating a list like above, the list sit’s within a panel. The panel body contains a series of DIVs, one for each list item, in this case with a class of notice.
To add the Status Alert, simply add the following classes to any div: status status-STATE. i.e.
<div class="notice status status-success">
  <a href="">
    <h6>Status Success</h6>
    <p>This is an example of a Status Success.</p>
  </a>
</div>

 

Table Status Alerts

When creating a table that contains Status Lights and Link Icons you will need to add certain classes.

To add Status Icons within a table, you should add .status-col to every <th> and <td> within that column. This will make that column 5% the table width and remove the excess padding from those cells. The Status Lights themselves should be placed within a <div> tag, within the relative <td>.
i.e: <td class="status-col"> <div class="status status-alert"></div> </td>.

Variables Columns

Bootstap tables spread flexible columns evenly across the table. Sometimes, for purposes of layout and legibility you may want to set a column or multiple columns to be a set width. To do this we have come up with Variable column widths for a table.

Simply add a class of width you require to the top column TH or TD and that column will set to that width, all others will adjust to fill. If you wish to completely control the width of all columns then the widths must add up to 100

Text Links

To include Link Icons before the text link so that the icon hangs properly on multi-line links, Add the class class="link-arrow".

When in a List within a Panel, it can be added to the UL tag <ul class="link-arrow list-group"></ul>.

When in a Table it should be added to the individual TD tag, and the link should be wrapped in a p tag,<td class="link-arrow"></td>

In all cases the link should be wrapped in a p tag. ie:
<ul class="link-arrow list-group">
    <li><p><a href="#">List Link</a></p></li>
</ul>
or
<td class="link-arrow"><p><a href="#">Table Link</a></p>

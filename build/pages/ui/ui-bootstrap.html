Tables

Tables should always sit within Panels, even if those panels have no heading or footer. They will however need to sit within a Panel Body.

 

Best Practice

Full width Table within Panels

To make the table to extend the full width of the panel then you should ADD a class of .panel-table to the Panel-Body that contains that table.

Responsive Tables

To make the table responsive the table needs to be wrapped with a .table-responsive class.
In most cases that class should be added to the containing panel. i.e:
<div class="panel panel-default table-responsive">.

Popovers

Popovers are added by using a link with NO url, like this:
<a class="pull-right glyphicon glyphicon-info-sign" data-toggle="popover" title="Add popover title here" data-placement="bottom" data-content="Add popover content here."></a>

Tooltips

Tooltips are added by adding the following:
<a class="pull-right glyphicon glyphicon-info-sign" data-toggle="tooltip" title="This is an example of a tootip."></a>

Tooltips are activated by hovering over them while Popovers are activated by clicking on them.

Alerts

Alerts do NOT go inside a ROW

If however you need to place it in a ROW then add row-indent to then row class so that it aligns correctly.

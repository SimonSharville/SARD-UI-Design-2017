Variables Columns

Bootstap tables spread flexible columns evenly across the table. Sometimes, for purposes of layout and legibility you may want to set a column or multiple columns to be a set width. To do this we have come up with Variable column widths for a table.

Simply add a class of width you require to the top column TH or TD and that column will set to that width, all others will adjust to fill. If you wish to completely control the width of all columns then the widths must add up to 100

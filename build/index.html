Style Guides

Welcome to the SARD Appraisal System Style Guide

Every effort has been made to be as clear as possible, but if you notice anything missing or have any questions, please contact me at design@simonsharville.co.uk

The system is built on the Bootstrap framework and wherever possible uses their standard components styled to fit the SARD identity. When this is not possible we have created a SARD specific component, these have a SARD specific sub-class added. i.e. a bootstrap Alert will have a class of class="alert alert-success" while a SARD specific Alert will have a class of class="alert alert-MRv".

For this wireframe we haven't made all the navigation as live links, instead click on any link below to visit that page, to return to this menu just click on the the Header image.
